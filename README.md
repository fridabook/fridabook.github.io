# 解密 Frida

> 动态插桩框架源码之旅

一本深度解析 [Frida](https://github.com/frida/frida) 动态插桩框架源码的技术书籍。

## 内容概览

全书分为 9 个部分，共 30 章 + 3 个附录：

1. **起步篇**（第1-3章）：Frida 是什么、源码全景、语言基础
2. **核心架构篇**（第4-7章）：入口分析、Gum 引擎、数据结构、构建系统
3. **插桩引擎篇**（第8-11章）：Interceptor、Stalker、代码生成器、Relocator
4. **注入与代理篇**（第12-15章）：进程注入、Agent 生命周期、脚本引擎、Gadget
5. **平台子系统篇**（第16-19章）：Linux、Darwin、Windows、移动平台
6. **通信与协议篇**（第20-22章）：DBus、控制服务、消息系统
7. **绑定与工具篇**（第23-25章）：Python、Node.js、CLI 工具
8. **高级话题篇**（第26-28章）：内存操作、跨平台设计、性能优化
9. **总结篇**（第29-30章）：设计模式总结、读者下一步

## 构建

```bash
# 构建 HTML 并在浏览器中打开
make

# 启动本地预览服务器
make serve

# 清理构建产物
make clean
```
